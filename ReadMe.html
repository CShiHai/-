SUPPLEMENTARY MATERIAL for

Penalized Orthogonal Iteration for Sparse Estimation of Generalized Eigenvalue Problem 

by Jung, Ahn and Jeon


Supplementary material includes two items.

1. SpGEPsupp.pdf: Document containing an extensive list of statistical GEP problems and additional numerical results.

2. seig.zip: Matlab functions that perform the sparse generalized eigenvector estimation as described in the article.
